# Aarya StockSense Pro 📈
**Institutional signals. Personal scale.**

Free, offline-capable trading dashboard — US, India, Crypto, UK, Europe, Canada, Japan.

## ⚡ Launch (Windows)
1. Install **Python 3.10+** from python.org  (tick "Add to PATH")
2. Double-click **`run.bat`** — installs packages on first run, opens browser automatically

## 📁 Files
| File | Purpose |
|------|---------|
| `app.py` | 6-tab Streamlit UI |
| `engine.py` | Strategy engine — signals, R/R, fundamentals, news, options, compounding |
| `config.py` | 7 market configs + settings persistence |
| `run.bat` | Windows one-click launcher |
| `requirements.txt` | Python dependencies |

## 🗂️ Tabs
1. **Today's Picks** — Screener with overview table, signal cards, fundamentals, news, charts
2. **AI Copilot** — 7 analyst personas: Hedge Fund, Value, Swing, Earnings, ETFs, Weekly DCA, Global
3. **Mutual Funds** — Fund catalogue + 30-year compounding simulator (nominal vs real)
4. **Stock Checker** — Any ticker: verdict + win probability + bull/base/bear targets + options
5. **My Portfolio** — Live P&L, stop alerts, EMA hold status, T1/T2 tracking
6. **Settings** — Custom watchlists per market + risk parameters
